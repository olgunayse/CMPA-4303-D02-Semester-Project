Welcome

This project is a simple tool that helps users compare two homes. It focuses on price, location, and size so the comparison is easier to understand.

The main feature of this project is on the Compare page. The user can choose what matters most and then see which home may be the better fit.

Go to the Compare page

What this project does

  • Shows two homes side by side
  • Helps users compare price, location, and size
  • Uses a simple interactive feature to update the result
