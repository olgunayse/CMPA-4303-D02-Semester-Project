# Smarter Comparisons

## What is this?
Compare Homes Smarter is a simple web project that helps users compare two homes. The project focuses on making home comparisons easier to understand by showing price, location, size, and trade-offs all in one place.

## Why does it exist?
I wanted to create a project related to real estate. Buyers usually compare homes using more than just one factor, so I liked the idea of building something that organizes those differences in a much simpler and clearer way.

## What tools did I use?
- HTML was used to structure the pages and organize the content.
- CSS was used to create a consistent design and layout across the site, while also helping with responsiveness.
- JavaScript was used to build the interactive comparison tool so users can input values and receive a recommendation.
- GitHub and GitHub Pages were used to store the project and deploy it as a live website.
