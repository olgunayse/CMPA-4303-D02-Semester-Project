Home Compare

Semester Project

Compare two homes in one simple view

This project is a basic comparison tool that helps users look at two homes side by side. Instead of jumping between listings, the user can focus on the main differences in price, space, and location in one place.

Start comparing

What this project focuses on

  • clear side by side comparison
  • simple design that is easy to follow
  • one useful interactive feature

Why this tool is useful

It puts the important details together

A buyer can see two homes at the same time instead of trying to remember details from separate pages.

It makes trade offs easier to understand

One home might have more space, while another might have a better location. This layout makes those differences easier to notice.

It keeps the process simple

The goal of the project is not to overload the user. It is meant to stay clear, readable, and practical.

What gave me the idea

I wanted to build something that felt useful and connected to real estate. Buyers usually compare homes by more than just price, so I liked the idea of creating a tool that shows several factors at once without making the page feel too busy.
