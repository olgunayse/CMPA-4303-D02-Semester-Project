About the Project

Why I built this

I built this tool to address a common problem in real estate: decision paralysis. When looking at multiple home listings, it is easy to get overwhelmed by the raw data. This project simplifies that data into a comparative format that emphasizes trade-offs.

By focusing on buyer priorities rather than just listing features, I wanted to create a more user-centric experience that feels practical and professional.
