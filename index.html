Welcome to Compare Homes Smarter

This project helps users compare two homes by looking at price, location, size, and buyer priorities in one clear view.

What this project does

Instead of looking at separate listings, users can compare two homes side by side and better understand the trade offs between them.

Main feature

On the Compare page, the user can choose what matters most and receive a recommendation based on that priority.

Why it matters

  • Shows two homes in one place
  • Helps explain differences in price, space, and location
  • Uses an interactive feature that updates the result
