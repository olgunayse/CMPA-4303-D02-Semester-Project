# Smarter Comparisons

## What is this?
Compare Homes Smarter is a simple web project that helps users compare two homes. The project focuses on making home comparisons easier to understand by showing price, location, size, and trade-offs all in one place.

## Why does it exist?
I wanted to create a project related to real estate. Buyers usually compare homes using more than just one factor, so I liked the idea of building something that organizes those differences in a much simpler and clearer way.

## What tools did I use?
- HTML was used to structure the content because it provides a clear and organized layout for each page.
- CSS was used to create a consistent visual design and improve readability, especially across different screen sizes.
- JavaScript was used to build the interactive comparison feature, allowing users to input values and receive recommendations.
- GitHub and GitHub Pages were used to manage version control and deploy the project so it can be accessed publicly.
